初めてのGitの練習です。
基本的なワークフローを実践していきます。
.git add -> ファイルをステージングにアップ
.git commit -> ファイルをローカルリポジトリにアップ
.git rm -> ファイル削除をステージングエリアに反映
.git commit -v -> 変更点を表示してコミット
.git status -> 現在のコミットの状況
.git diff -> ファイルの差分を見る
.git log -> ファイルの変更履歴を見る
.git log --oneline -> ファイルの変更履歴を１行単位で見る
.git log -n 3 -> ファイルの変更履歴を３世代分だけ見る
.git log -p -> ファイルの変更履歴のなんというファイルの何を変更したのかがわかる
.git log -p first.txt -> first.txtというファイルに絞ってどこを変更したか履歴を見ることができる
※「j」で下に「k」で上にスクロールして見ることができる
